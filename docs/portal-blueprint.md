# QE Academy Portal Blueprint

## Purpose

QE Academy Portal is an educational application designed to teach QA Automation using Playwright and Python.

Students do not develop the portal.

Students automate the portal.

The portal must be fully functional before the academy starts.

---

## Roles

### Student

Can:

- Login
- View courses
- Search courses
- Filter courses
- View course details
- Enroll in courses
- Cancel enrollments
- View profile

### Instructor

Can:

- View courses
- View enrollments
- Manage courses

### Admin

Can:

- View courses
- Manage courses
- Manage users
