# QE Academy Portal

Educational training platform for QA Automation using Playwright and Python.

## Purpose

QE Academy Portal is a learning sandbox designed for QA Automation training.

Students will use the portal to practice:

- Playwright
- Assertions
- Locators
- Page Object Model (POM)
- Fixtures
- Test Data Management
- CRUD testing
- Role-based access testing
